# todolist
# 📝 To-Do List (Lista de Tarefas)

Este é um projeto simples de lista de tarefas feito com HTML, CSS e JavaScript.

## 🚀 Funcionalidades

- Adicionar tarefas
- Marcar como concluída ✔️
- Remover tarefas 🗑️
- Modo noturno 🌙
- Responsivo (funciona no celular)

---

## 📂 Estrutura

- index.html → estrutura da página
- style.css → estilo + modo escuro
- script.js → lógica do sistema

---

## 💡 Como funciona

### 1. HTML
Cria a estrutura da aplicação:
- Campo de input
- Botão de adicionar
- Lista de tarefas

Também usa:
- Bootstrap para layout
- Bootstrap Icons para ícones

---

### 2. CSS
Responsável pelo visual:

- Modo claro e escuro
- Responsividade
- Estilo das tarefas

Classe importante:
- `.completed` → marca tarefa como feita

---

### 3. JavaScript

Controla tudo:

#### addTask()
Adiciona uma nova tarefa na lista

#### removeTask()
Remove a tarefa

#### toggleComplete()
Marca ou desmarca como concluída

#### toggleDarkMode()
Alterna entre modo claro e escuro

---

## 📱 Responsividade

Usa Bootstrap + media queries para funcionar bem em celular.

---

## 🛠️ Como usar

1. Abra o VS Code
2. Crie os arquivos
3. Cole os códigos
4. Abra o `index.html` no navegador
